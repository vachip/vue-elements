# My Component library

Here's a brief introduction.

### Installation

$ npm install @vachip/vue-elements

### Usage

```js
import {createApp} from 'vue';
import vueElements from '@vachip/vue-elements'
import App from './App.vue'

createApp(App)
  .use(vueElements)
  .mount('#app')

```
